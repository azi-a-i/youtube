How it works

LLMNoteTube is built to move from topic to synthesis without hiding what is connected.

The product is intentionally split into clear stages so users always know whether anonymous research, NotebookLM access, and artifact generation are actually ready.

Stage 1

Start with anonymous research

Anyone can search YouTube by topic, review titles, views, channels, dates, and durations, then pick the strongest sources.

Stage 2

Connect NotebookLM when needed

When users want synthesis, they connect a NotebookLM browser session on the Connect page or directly inside the workspace.

Stage 3

Run the synthesis pipeline

The selected URLs move into NotebookLM, where the workspace can request summaries, comparisons, tactics, and deliverables.

Stage 4

Download outputs

Once the job finishes, users can review the analysis, inspect references, and download generated artifacts from one place.

Page structure

Overview introduces the product, Connect handles NotebookLM session setup, Workflow explains the path, and Workspace is where the full research run happens.

Important note

LLMNoteTube does not require a site account. NotebookLM still needs either a browser-provided session or a backend session configured by the deployment.
